# dotfiles

## dependencies:

* bspwm
* sxhkd
* alacritty
* starship
* polybar
* conky
* rofi
* picom
* nvim
* cozette-ttf

### Arch:
```
yay -S bspwm sxhdk alacritty starsip polybar conky rofi picom cozette-ttf
```
***
## install: 
make start.sh executable with
```
chmod +x start.sh
```
and run with
```
./start.sh
```
***

#### NOTE: qtile and kitty are opcional, unless you want to set them up, you can delete these folders.
